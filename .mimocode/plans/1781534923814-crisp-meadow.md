# Plan: Guided Tour of TradeScape in Roblox Studio

## Goal
Walk the user through the game visually in Roblox Studio so they understand the full architecture and each screen.

## Step-by-Step Guide

### Step 1: Open and Play
- Verify Studio is open with `tradescape.rbxlx`
- Press **F5** (or click Play ▶) to start the game
- The player spawns with $10,000 virtual balance

### Step 2: Market Screen (Main UI)
- Left panel: Watchlist with 16 symbols (AAPL, TSLA, MSFT, etc.)
- Each row shows: symbol, name, price, change %, volume
- Prices refresh every 2 seconds
- Green = up, Red = down
- Bloomberg dark theme

### Step 3: Buy a Stock
- Click on any stock (e.g., AAPL)
- TradeWidget modal opens
- Enter shares (e.g., 10)
- Click "Buy"
- Confirmation: "Bought 10 AAPL at $296.80. Fee: $3"
- Balance decreases, position appears

### Step 4: Portfolio View
- After buying, see your position in the portfolio section
- Shows: shares, avg price, current price, P&L, profit %
- Real-time updates

### Step 5: Key Shortcuts
- **Tab** → Profile (XP, level, perks, missions)
- **C** → Club panel
- **L** → Leaderboard
- **S** → Shop

### Step 6: Profile Screen (Tab)
- XP bar showing progress to next level
- Current rank (Novato → Trader → Broker → Magnate → Whale)
- Perks tab: available upgrades
- Missions tab: daily challenges
- Office info: current tier

### Step 7: Charts (Click a stock → Chart)
- Candlestick chart with OHLCV data
- Timeframes: 1D, 1W, 1M, 3M, 6M, 1Y
- Volume bars at bottom

### Step 8: News Widget
- "Why did it move?" contextual news
- Real headlines from Yahoo Finance
- Educational tips

### Step 9: Social Features
- **C key** → Club screen (create/join trading clubs)
- **L key** → Global leaderboard (by value, profit %, level)

### Step 10: Shop (S key)
- 6 Robux products (cosmetics only)
- Office skins, name colors, badges

## Files Involved
- All files in `roblox/StarterPlayer/StarterPlayerScripts/UI/` (client UI)
- All files in `roblox/ServerScriptService/` (server logic)
- `roblox/ServerScriptService/NetworkHandler.server.lua` (RemoteFunction handlers)

## Verification
- User sees the market UI with live prices
- User can buy/sell stocks
- User sees P&L in portfolio
- User can navigate with Tab/C/L/S keys
