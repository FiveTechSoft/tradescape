# 🏦 TradeScape

Simulador de trading RPG en Roblox con datos reales de mercados globales. El mercado real es el juego.

## Stack

- **Juego:** Roblox (Luau)
- **Backend:** Node.js + Express + `yahoo-finance2`
- **Datos:** Yahoo Finance (gratis, global)
- **Hosting Proxy:** VPS $5-10/mes
